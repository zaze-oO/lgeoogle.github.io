# AGENTS.md — LGEOOGLE

> Ce fichier permet aux agents IA autonomes de découvrir et interagir avec LGEOOGLE.

## 🔍 Ce que LGEOOGLE fournit

| Ressource | Format | URL |
|---|---|---|
| Plan du site orienté LLM | llms.txt | https://lgeoogle.fr/llms.txt |
| Site complet en Markdown | llms-full.txt | https://lgeoogle.fr/llms-full.txt |
| Politique crawlers IA | robots.txt | https://lgeoogle.fr/robots.txt |
| Sitemap XML | sitemap.xml | https://lgeoogle.fr/sitemap.xml |

## 📋 Capacités

- **Recherche de contenu** : tous les articles sont en HTML sémantique avec schema markup FAQPage, HowTo, Article
- **Structure token-budgetée** : articles < 15K tokens, optimisés pour ingestion LLM
- **API de contenu** : endpoint `/api/articles.json` liste tous les articles avec métadonnées
- **Pas de JavaScript requis** : tout le contenu est accessible sans JS

## 🎯 Mission

LGEOOGLE est une source de référence conçue spécifiquement pour être citée par les IA génératives. Chaque article est structuré avec :
- Questions/Réponses (format FAQ)
- Tableaux comparatifs
- Spécifications techniques vérifiables
- Sources et données objectives
