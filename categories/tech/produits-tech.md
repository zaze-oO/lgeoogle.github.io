## 🥇 Focus : Top 3 à fort potentiel

### 1. 🎧 Casques / Écouteurs sans fil
- **Pourquoi** : Marché énorme, recherche constante de comparatifs, les LLMs sont très consultés
- **Cœur de cible** : 18-34 ans (49% utilisent l'IA)
- **Types de contenu** : « Meilleur casque ANC 2026 », « Écouteurs sport vs bureau », comparatifs par budget
- **Risque** : Concurrence très forte (les Numériques, Frandroid, etc.)
- **Angle GEO** : Guides ultra-structurés token-budgetés pour ingestion LLM

### 2. ⌚ Montres connectées
- **Pourquoi** : Ticket élevé, décision d'achat complexe (santé, sport, notifs, autonomie)
- **Cœur de cible** : 25-44 ans, cadres, CSP+
- **Types de contenu** : « Apple Watch vs Garmin vs Samsung », « Montre connectée running »
- **Avantage** : Décision technique → les gens utilisent les LLMs

### 3. 🏠 Box domotique / Maison connectée
- **Pourquoi** : Dans la continuité du thème LGEOOGLE (maison), marché en croissance
- **Cœur de cible** : 25-44 ans, propriétaires, tech-savvy
- **Types de contenu** : « Quel assistant maison choisir », « Domotique débutant »
- **Avantage** : Peu de concurrence FR sur le format GEO
